As páginas a seguir fornecem várias técnicas para melhorar a acessibilidade nos formulários web:

  • Dicas básicas sobre formulários: Adicionando dicas e descrições para campos inválidos, ou obrigatórios
  • Alertas: Usando alertas para fornecer mensagens de erro de validação no lado cliente
  • Rótulos de múltiplas partes: Habilitando rótulos (labels) para formulários complexos, com um controle para cada campo

Ver também o Artigo do Yahoo! sobre validação de formulário e ARIA, cobrindo muito do mesmo conteúdo.
