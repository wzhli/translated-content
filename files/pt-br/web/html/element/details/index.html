Sumário

O elemento HTML details (<details>) é usado como uma ferramenta de onde o usuário irá obter informações adicionais.

Contexto de uso

Conteúdo permitido Um elemento {{ HTMLElement("summary") }} seguido por flow content
Omissão de tag Nenhuma, tanto as tas iniciais quanto as finais são obrigatórias.
Elementos pais permitidos Qualquer elemento que aceite flow content.
Documento normativo HTML5, section 4.11.1

Atributos

Como todos os elementos HTML, esse elemento aceita os global attributes.

{{ htmlattrdef("open") }}
Esse atributo Booleano indica se os detalhes serão mostrados para o usuário ao carregar a página. Se omitido os detalhes não serão mostrados.

Interface do DOM

Esse elemento implemena a interface HTMLDetailsElement.

Exemplos

<details>
  <summary>Alguns detalhes</summary>
  <p>Mais informações sobre os detalhes.</p>
</details>
