Pode conter somente texto e quaisquer marcações contidas no texto não são interpretadas.</p> <ul class="htmlelt"> <li><dfn><a href="/en-US/docs/HTML/Content_categories" title="HTML/Content_categories">Categorias de conteúdo</a></dfn><a href="/pt-BR/docs/Web/Guide/HTML/Content_categories#Metadata_content">Conteúdo de metadados</a>.</li> <li><dfn>Conteúdo permitido</dfn> Texto que não seja whitespace entre elementos.</li> <li><dfn>Omissão de marcação</dfn>Todas as marcações são necessárias.
