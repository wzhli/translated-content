Resumo

O Elemento HTML <legend> (ou Elemento HTML Campo "Legend") representa um rótulo para o conteúdo do seu ancestral {{HTMLElement("fieldset")}}.

  • Categorias de Conteúdo Nenhuma.
  • Conteúdo permitido Conteúdo Textual.
  • Omissão de Tag {{no_tag_omission}}
  • Elementos ancestrais permitidos Um {{HTMLElement("fieldset")}} cujo primeiro filho seja esse elemento <legend>
  • Interface DOM {{domxref("HTMLLegendElement")}}

Atributos

Esse elemento apenas inclui os atributos globais.

Exemplos

Veja {{HTMLElement("form")}} para exemplos sobre <legend>.
