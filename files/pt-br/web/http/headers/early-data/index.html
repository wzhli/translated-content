O cabeçalho Early-Data é colocado por um intermediário para indicar que a requisição foi transportada em dados precoces do TLS (TLS early data), e também indica que o intermediário entende o código de status {{HTTPStatus("425", "425 (Too Early)")}}.

O cabeçalho Early-Data não é definido pelo gerador da requisição (i.e., um navegador).

Tipo de cabeçalho {{Glossary("Request header")}}
{{Glossary("Forbidden header name")}} não

Sitnaxe

Early-Data: 1

Exemplos

GET /resource HTTP/1.0
Host: example.com
Early-Data: 1
