La méthode **`Window.home()`** renvoie la fenêtre à la page d'accueil.

## Syntaxe

    window.home();

## Exemple

```js
function goHome() {
  window.home();
}
```

## Spécification

DOM Niveau 0. Ne fait pas partie de la spécification.
