_Un nom d'en-tête de réponse interdit est un_ nom d'[en-tête HTTP](/fr/docs/HTTP/Headers) (\``Set-Cookie`\` ou \``Set-Cookie2`\`) qui ne peuvent être modifiés par programmation.
