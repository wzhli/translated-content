静的型付け言語（例えば、Java, C, または C++ ）とは、コンパイル時に変数の型がわかる言語です。静的型付け言語の多くは、プログラマが明示的に型を指定しなければなりませんが、OCamlのように型推論によってプログラマが変数の型を指定しなくてもよい場合もあります。
