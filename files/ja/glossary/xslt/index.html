eXtensible Stylesheet Language Transformations (XSLT) は、{{Glossary("XML")}} 文書を他の XML 文書に変換するために使用される宣言的言語であり、{{Glossary("HTML")}}、{{Glossary("PDF")}}、プレーンテキストなどが含まれます。

XSLT には XML 入力、または XQuery および XPath データ・モデルに変換可能な任意のフォーマットを受け入れる独自のプロセッサーがあります。XSLT プロセッサは XML ドキュメントと XSLT スタイルシートに基づいて新しいドキュメントを作成し、プロセス内の元のファイルに変更を加えません。
