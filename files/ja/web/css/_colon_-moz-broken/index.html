:-moz-broken は CSS の擬似クラスで、 Mozilla 拡張 であり、壊れた画像リンクを表している要素に一致します。

注: このセレクターは、主にテーマ開発者が使用することを意図しているものです。

構文

:-moz-broken

例

HTML

<img src="broken.jpg" alt="This image is broken. :-(">

CSS

:-moz-broken {
  background: bisque;
  padding: 8px;
}
