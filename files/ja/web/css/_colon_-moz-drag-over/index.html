CSS の :-moz-drag-over 擬似クラスは Mozilla 拡張で、 {{event("dragover")}} イベントが呼び出される時に要素に一致します。

構文

:-moz-drag-over

例

HTML

<table border="1">
  <tr>
    <td width="100px" height="100px">Drag Over</td>
  </tr>
</table>

CSS

td:-moz-drag-over {
  color: red;
}
