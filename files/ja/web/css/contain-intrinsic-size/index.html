{{CSSRef}}

contains-intrinsic-size CSSプロパティは、content-visibilityで指定された要素の自然なサイズを制御します。

Syntax

/* <length> values */
contain-intrinsic-size: 1000px;
contain-intrinsic-size: 10rem;

/* <percentage> value */
contain-intrinsic-size: 10%;
