{{APIRef}}

dragstart イベントは、ユーザーが要素や選択されたテキストをドラッグし始めたときに発生します。

バブリング あり
キャンセル可能 はい
既定のアクション ドラッグ＆ドロップ操作を始める。
インターフェイス {{domxref("DragEvent")}}
イベントハンドラープロパティ {{domxref("GlobalEventHandlers/ondragstart", "ondragstart")}}
