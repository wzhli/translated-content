**`GeolocationCoordinates.latitude`** は読み取り専用のプロパティであり、 `double` 値でその位置の緯度を 10 進形式で表します。

## 構文

```js
let lat = geolocationCoordinatesInstance.latitude
```

### 値

`double` 値でその位置の緯度を 10 進形式で表します。
