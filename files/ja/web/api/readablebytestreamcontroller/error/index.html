{{domxref("ReadableByteStreamController")}} インターフェイスの error() メソッドにより、関連するストリームとの今後のやり取りがエラーになります。

構文

readableByteStreamController.error(e);

パラメーター

e
今後のやり取りを失敗させるエラー。

戻り値

undefined。

例外

TypeError
ソースオブジェクトが ReadableByteStreamController ではないか、何らかの理由でストリームが読み取り可能ではありません。

例

未定。
