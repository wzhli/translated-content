WebVR API の vrdisplaypresentchange イベントは，VRデバイスの表示状態が変化したときに発火します — つまり，表示状態から非表示になったとき，あるいはその逆のケースです．

Bubbles No
Cancelable No

Interface {{domxref("Event")}}
