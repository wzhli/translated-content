{{APIRef("DOM Events")}}

**`offsetY`** は {{domxref("MouseEvent")}} インターフェイスの読み取り専用プロパティで、このイベントと対象ノードのパディング辺との間のマウスポインターの Y 座標におけるオフセットを提供します。

## 値

`double` 型の浮動小数点値です。

仕様書の早期の版では、整数として定義されていました。
