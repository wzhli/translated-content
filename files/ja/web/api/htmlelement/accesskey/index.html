Element.accessKeyは、ユーザが他の要素へジャンプする時に押す、キーストロークを設定します。
 

Element.accessKey は、ブラウザの既定のキーバインディングと競合する為、めったに使われません。この競合を回避するために、ブラウザはアクセスキーの挙動を他の「最適な」キーと一緒に押されたときに動くように実装しています。（ Alt + アクセスキー、の様に。）
