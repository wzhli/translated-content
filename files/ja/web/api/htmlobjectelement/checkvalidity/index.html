{{APIRef("HTML DOM")}}

checkValidity() は {{domxref("HTMLObjectElement")}} インターフェイスのメソッドで、常に true である {{jsxref("Boolean")}} を返します。 object オブジェクトは制約の検証の候補にはならないからです。

構文

const valid = HTMLObjectElement.checkValidity();

引数

なし。

返値

true

例外

なし。
