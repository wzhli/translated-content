tag は {{domxref("Notification")}} インターフェイスの読取専用プロパティで、 {{domxref("Notification.Notification","Notification()")}} コンストラクターの tag オプションで指定された通知の識別タグを示します。

通知タグの考え方は、複数の通知が同じタグを共有し、それらを一緒にリンクすることです。ユーザーの画面が膨大な数の類似した通知で埋め尽くされるのを避けるために、一つの通知をプログラム的に別のものと置き換えることができます。

構文

var tag = Notification.tag;

値

{{domxref("DOMString")}} です。

例

通知 API の使用の記事に、タグの使い方の良い例があります。
