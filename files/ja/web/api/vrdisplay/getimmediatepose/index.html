{{domxref("VRDisplay")}} インターフェイスの getImmediatePose() メソッドは，予測が適用されていない状態の VRDisplay の現在のポーズを決める  {{domxref("VRPose")}} オブジェクトを返します．

シンタックス

var myImmediatePose = vrDisplayInstance.getImmediatePose();

パラメータ

なし．

戻り値

{{domxref("VRPose")}} オブジェクト．
