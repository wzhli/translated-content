SVG の構文と使い方の基本的な例は、 W3C SVG test suite にあります。

注: Gecko 2.0 より、Gecko は SMIL を用いた SVG アニメーションに対応しています。簡単な概要についてはSMIL を用いた SVG アニメーションを参照してください。 SVG の完全なドキュメントは近日公開予定です。いつの日か。

SVG 2 の変更についての Mozilla の対応の解説もあります。

要素の実装状況

SVG 1.1 の要素とネイティブサポートの状況の概略です。

  • SVGSVGElement
