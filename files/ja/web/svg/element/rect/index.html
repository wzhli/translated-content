概要

rect 要素はSVGの基本図形であり、四角形を生成する際の基準となる座標と、そこからの幅と高さを指定します。また、角の丸まった四角形を作ります。

例

シンプルな rect 要素の使い方

» rect-1.svg

角の丸い rect 要素

» rect-2.svg
