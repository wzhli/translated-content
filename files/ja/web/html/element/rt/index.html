HTML のルビ文字列 (<rt>) 要素は、ルビによる注釈（振り仮名）のルビ文字列の部分を定義し、東アジアの組版において発音、翻訳、音写などの情報を提供するために使用します。 <rt> 要素は常に {{HTMLElement("ruby")}} 要素の中で使用されます。

より多くの例については {{HTMLElement("ruby")}} 要素についての記事を参照してください。

属性

この要素にはグローバル属性のみがあります。

例

この例では、 {{HTMLElement("ruby")}} 要素内の漢字にローマ字表記を提供します。

<ruby>
  漢 <rt>Kan</rt>
  字 <rt>ji</rt>
</ruby>

body {
  font-size: 22px;
}
