{{ XsltRef() }}


math:highest() 指定されたノードセットの最大値を持つノードを返却する。( math:max() を使用して計算した最大値).

ノードの文字列を数値に変換した場合、一つのノードが最大値を持つ。

{{ Note() }}

構文 

math:highest(nodeSet)

引数

nodeSet
最大値を持つノードが返されるノードセット

戻り値

math:max()が返したノードのコピーからなる 結果ツリー・フラグメント

定義

EXSLT - MATH:HIGHEST

Gecko 対応

Gecko 1.9、および以降でサポート
