{{HTTPSidebar}}

HTTP の GET メソッドは、特定のリソースの表現をリクエストします。 GET を使用したリクエストはデータを受け取るだけです。

リクエストの本文 なし
成功時のレスポンスの本文 あり
{{Glossary("Safe", "安全性")}} あり
{{Glossary("Idempotent", "べき等性")}} あり
{{Glossary("Cacheable", "キャッシュ")}} 可
HTML フォームでの使用 可

構文

GET /index.html
