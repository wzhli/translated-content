Der HTTP-Statuscode 418 I'm a teapot zeigt an, dass der Server sich weigert, Kaffee zu kochen, weil er ein Teekessel ist. Diese Fehlermeldung bezieht sich auf das Hyper Text Coffee Pot Control Protocol, einen Aprilscherz aus 1998.
