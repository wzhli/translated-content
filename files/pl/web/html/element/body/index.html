{{HTMLRef}}

Element <body> reprezentuje zawartość dokumentu HTML. W dokumencie może znajdować się tylko jeden element <body>.

Kategorie treści Podział na sekcje.
Dozwolona zawartość Zawartość przepływu.
Pominięcie znacznika Znacznik otwierający może zostać pominięty, jeśli pierwszą rzeczą wewnątrz niego nie jest znak spacji, komentarz, element {{HTMLElement("script")}} lub element {{HTMLElement("style")}}. Znacznik zamykający może zostać pominięty, jeśli element <body> ma zawartość lub ma znacznik startowy, a po nim nie następuje od razu komentarz.
Dozwoleni rodzice To musi być drugi element elementu {{HTMLElement("html")}}.
Dozwolone role ARIA Brak
Interfejs DOM {{domxref("HTMLBodyElement")}}
  • Element <body> wyświetla interfejs {{domxref("HTMLBodyElement")}}.
  • Dostęp do elementu <body> można uzyskać poprzez właściwość {{domxref("document.body")}}.

Atrybuty

Ten element uwzględnia atrybuty globalne.

{{htmlattrdef("alink")}} {{obsolete_inline}}
Kolor tekstu dla hiperłączy po zaznaczeniu. Ta metoda jest nieprawidłowa, użyj właściwości CSS {{cssxref("color")}} w połączeniu z pseudo-klasą {{cssxref(":active")}}.
{{htmlattrdef("background")}} {{obsolete_inline}}
URI obrazu do wykorzystania jako tło. Ta metoda jest nieprawidłowa, zamiast tego należy użyć właściwości CSS {{cssxref("background")}} na elemencie.
{{htmlattrdef("bgcolor")}} {{obsolete_inline}}
Kolor tła dokumentu. Ta metoda jest nieprawidłowa, użyj zamiast tego właściwości CSS {{cssxref("background-color")}} na elemencie.
{{htmlattrdef("bottommargin")}} {{obsolete_inline}}
Margines dolnej części strony. Ta metoda jest nieprawidłowa, zamiast tego użyj właściwości CSS {{cssxref("margin-bottom")}} na elemencie.
{{htmlattrdef("leftmargin")}} {{obsolete_inline}}
Margines lewej części strony. Ta metoda jest nieprawidłowa, zamiast tego użyj właściwości CSS {{cssxref("margin-left")}} na elemencie.
{{htmlattrdef("link")}} {{obsolete_inline}}
Kolor tekstu dla nie odwiedzonych linków hipertekstowych. Ta metoda jest nieprawidłowa, zamiast tego użyj właściwości CSS {{cssxref("color")}} w połączeniu z pseudo-klasą {{cssxref(":link")}}.
{{htmlattrdef("onafterprint")}}
Funkcja do wywołania po wydrukowaniu dokumentu przez użytkownika.
{{htmlattrdef("onbeforeprint")}}
Funkcja do wywołania, gdy użytkownik żąda wydrukowania dokumentu.
{{htmlattrdef("onbeforeunload")}}
Funkcja do wywołania, gdy dokument jest w trakcie ładowania.
{{htmlattrdef("onblur")}}
Funkcja do wywołania, gdy dokument straci ostrość.
{{htmlattrdef("onerror")}}
Funkcja do wywołania, gdy dokument nie zostanie załadowany prawidłowo.
{{htmlattrdef("onfocus")}}
Funkcja do wywołania, gdy dokument uzyska ostrość.
{{htmlattrdef("onhashchange")}}
Funkcja do wywołania w przypadku zmiany części identyfikatora fragmentu (rozpoczynającego się znakiem hash ('#')) bieżącego adresu dokumentu.
{{htmlattrdef("onlanguagechange")}} {{experimental_inline}}
Funkcja do wywołania po zmianie preferowanych języków.
{{htmlattrdef("onload")}}
Funkcja do wywołania po zakończeniu ładowania dokumentu.
{{htmlattrdef("onmessage")}}
Funkcja do wywołania, gdy dokument otrzyma wiadomość.
{{htmlattrdef("onoffline")}}
Funkcja do wywołania, gdy zawiodła komunikacja sieciowa.
{{htmlattrdef("ononline")}}
Funkcja do wywołania po przywróceniu komunikacji sieciowej.
{{htmlattrdef("onpopstate")}}
Funkcja do wywołania po przejściu użytkownika do historii sesji.
{{htmlattrdef("onredo")}}
Funkcja do wywołania, gdy użytkownik przeszedł do przodu w historii operacji cofania.
{{htmlattrdef("onresize")}}
Funkcja do wywołania po zmianie rozmiaru dokumentu.
{{htmlattrdef("onstorage")}}
Funkcja do wywołania, gdy obszar pamięci masowej ulegnie zmianie.
{{htmlattrdef("onundo")}}
Funkcja do wywołania po cofnięciu się użytkownika w historii operacji.
{{htmlattrdef("onunload")}}
Funkcja do wywołania, gdy dokument wygasa.
{{htmlattrdef("rightmargin")}} {{obsolete_inline}}
Margines prawej części strony. Ta metoda jest nieprawidłowa, zamiast tego należy użyć właściwości CSS {{cssxref("margin-right")}} na elemencie.
{{htmlattrdef("text")}} {{obsolete_inline}}
Kolor tekstu na pierwszym planie. Ta metoda jest nieprawidłowa, należy użyć właściwości CSS {{cssxref("color")}} na elemencie.
{{htmlattrdef("topmargin")}} {{obsolete_inline}}
Margines górnej części strony. Ta metoda jest nieprawidłowa, należy użyć właściwości CSS {{cssxref("margin-top")}} na elemencie.
{{htmlattrdef("vlink")}} {{obsolete_inline}}
Kolor tekstu dla odwiedzonych linków hipertekstowych. Ta metoda jest nieprawidłowa, należy użyć właściwości CSS {{cssxref("color")}} w połączeniu z pseudo-klasą {{cssxref(":visited")}}.

Przykład

<html>
  <head>
    <title>Tytuł dokumentu</title>
  </head>
  <body>
    <p>To jest akapit</p>
  </body>
</html>
