{{APIRef("HTML DOM")}}

El método HTMLMediaElement.pause() pausará la reproducción de los medios, si los medios ya están en pausa, este método no tendrá efecto.

Sintaxis

HTMLMediaElement.pause()

Parametros

Ninguno.

Valor de retorno

Ninguno.

Excepciones

Ninguno.
