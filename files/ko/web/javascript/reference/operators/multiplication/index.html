곱하기 연산자(*)는 두 연산자의 곱을 구합니다.

{{EmbedInteractiveExample("pages/js/expressions-multiplication.html")}}

구문

x * y

예제

곱셈

 2 * 2     // 4
-2 * 2     // -4

Infinity 곱하기

Infinity * 0         // NaN
Infinity * Infinity  // Infinity

숫자가 아닌 경우

'foo' * 2 // NaN
