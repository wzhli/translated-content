나누기 할당 연산자(/=)는 오른쪽 피연산자의 값으로 변수를 나눈 결과를 다시 변수에 할당합니다.

{{EmbedInteractiveExample("pages/js/expressions-division-assignment.html")}}

구문

x /= y // x = x / y

예제

나누기 할당 사용하기

// bar = 5
// 위와 같은 변수를 가정하고, 아래의 모든 연산을 순서대로 실행할 때

bar /= 2     // 2.5
bar /= 2     // 1.25
bar /= 0     // Infinity
bar /= 'foo' // NaN
