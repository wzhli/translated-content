클라이언트 리디렉션 응답 코드 304 Not Modified는 요청된 리소스를 재전송할 필요가 없음을 나타낸다. 캐시된 자원으로의 암묵적인 리디렉션이다. 이 는 {{HTTPMethod("GET")}}나 {{HTTPMethod("HEAD")}} 요청처럼 요청 방법이 {{glossary("안전")}}한 경우 또는 요청이 조건부로 {{HTTPHeader("If-None-Match")}} 또는 {{HTTPHeader("If-Modified-Since")}} 헤더를 사용할 때 응답 된다.

이에 상응하는 {{HTTPStatus("200")}} OK 응답에는 {{HTTPHeader("Cache-Control")}}, {{HTTPHeader("Content-Location")}}, {{HTTPHeader("Date")}}, {{HTTPHeader("ETag")}}, {{HTTPHeader("Expires")}}, 그리고 {{HTTPHeader("Vary")}} 가 포함되어 있었을 것이다.
 

브라우저의 개발자도구 네트워크 패널은 304 응답으로 이어지는 많은 요청을 생성하며, 로컬 캐시로 액세스 하는 것을 개발자에게 보여준다.
