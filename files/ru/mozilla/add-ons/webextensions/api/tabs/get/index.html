{{AddonSidebar()}}

Получает сведения об указанной вкладке.

Синтаксис

browser.tabs.get(
  tabId,              // integer
  function(tab) {...} // function
)

Параметры

tabId
integer.
callback
function. Функция принимает следующие аргументы:
tab
{{WebExtAPIRef('tabs.Tab')}}.

Совместимость с браузерами

{{Compat}}

{{WebExtExamples}}
