Заголовок Content-Length указывает размер отправленного получателю тела объекта в байтах.

Тип заголовка {{Glossary("Entity header", "Заголовок сущности")}}
Можно не передавать да

Синтаксис

Content-Length: <длина>

Директивы

<длина>
Байты.
