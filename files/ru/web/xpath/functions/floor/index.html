  Функция floor принимает десятичное число и возвращает наибольшее целое, меньшее или равное исходному.

Синтаксис

floor( number )

Аргументы

number
Десятичное число

Возвращаемое значение

Ближайшее целое, меньшее или равное number.

Определено

XPath 1.0 4.4
