Тип String
Обязателен Нет
Пример 
"short_name": "Awesome app"

Элемент short_name -- это строка, которая определяет название веб-приложения показываемое пользователю, если для отображения name не достаточно места (например, в качестве надписи под иконкой приложения на экране телефона). short_name может отображаться слева-направо или справе-налево, в зависимости от значений элементов манифеста dir and lang.

Примеры

Обычно short_name отображается слева-направо:

"name": "Awesome application",
"short_name": "Awesome app"

short_name на арабском отображается справа-налево:

"dir": "rtl",
"lang": "ar",
"name": "تطبيق رائع",
"short_name": "رائع"
