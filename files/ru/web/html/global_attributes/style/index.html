{{HTMLSidebar("Global_attributes")}}

Глобальный атрибут стиля содержит объявления стилей CSS, которые могут применяться к элементу. Обратите внимание: рекомендуется, чтобы стили были определены в отдельном файле или файлах. Этот атрибут и элемент {{HTMLElement ("style")}} имеют в основном назначение для быстрого добавления стиля, например, для целей тестирования.

Замечание об использовании: этот атрибут не должен использоваться для передачи семантической информации. Даже если весь стиль удаляется, страница должна оставаться семантически корректной. Обычно он не должен использоваться для скрытия несущественной информации; это должно быть сделано с использованием скрытого атрибута.
