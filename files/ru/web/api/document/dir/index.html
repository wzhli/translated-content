Свойство Document.dir является строкой {{domxref("DOMString")}} показывает направление текста на странице (слева направо или справа налево).

Возможные значения:

  • 'ltr' - слева направо

  • 'rtl' - справа налево

Синтаксис

dirStr = document.dir;
document.dir = dirStr;
