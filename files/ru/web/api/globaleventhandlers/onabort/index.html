Summary

Обработчик событий, позволяющий прерывать события объекта window. (Недоступен для Firefox 2 и Safari).

Syntax

window.onabort = funcRef
  • funcRef: Reference to a function

Example

window.onabort = function() {
  alert("Load aborted.");
}
