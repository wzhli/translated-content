Используйте Element.createShadowRoot чтобы создать  экземпляр shadow DOM. После создания shadow DOM, он всегда будет привязан к существующему элементу. После создания shadowDOM, элемент к которому он привязан будет называться Теневой корень ({{glossary("shadow root","shadowRoot")}}) .

This method has been deprecated in favor of attachShadow.

Синтаксис

var shadowroot = element.createShadowRoot();

Отсутствуют

Значение результата

Возвращает {{domxref("ShadowRoot")}}.
