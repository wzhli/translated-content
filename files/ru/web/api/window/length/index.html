Сводка

Возвращает кол-во фреймов (например {{HTMLElement("frame")}} или {{HTMLElement("iframe")}}) в текущем окне.

Синтаксис

framesCount = window.length;
  • framesCount - кол-во фреймов на странице.

Пример

if (window.length) {
  // Этот документ содержит фреймы
}
