{{APIRef("Console API")}}{{Non-standard_header}}

Добавляет маркер в браузерные инструменты Performance или Waterfall. Это помогает соотнести точку в вашем коде с другими записанными событиями.

{{AvailableInWorkers}}

Синтаксис

console.timeStamp(label);

Параметры

label
Название метки. Необязательный параметр.
