{{APIRef("SVG")}}

 SVGPathElement.getTotalLength() 该方法返回用户代理对路径总长度(以用户单位为单位)的计算值。

注意:在SVG 2中，该方法被移动到 {{DOMxRef("SVGGeometryElement")}} 接口，由 {{DOMxRef("SVGPathElement")}} 继承。

语法

float someElement.getTotalLength();

返回值

指示路径总长度(以用户单位为单位)的浮点数。
