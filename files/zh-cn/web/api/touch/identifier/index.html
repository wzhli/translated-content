概述

返回一个可以唯一地识别和触摸平面接触的点的值. 这个值在这根手指（或触摸笔等）所引发的所有事件中保持一致, 直到它离开触摸平面.

语法

var id = touchItem.identifier;

返回值

id
{{ domxref("Touch") }} 对象的唯一标识符.

标准定义

Touch Events Specification
