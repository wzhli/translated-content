概述

返回触点相对于可见视区(visual viewport)左边沿的的X坐标. 不包括任何滚动偏移.这个值会根据用户对可见视区的缩放行为而发生变化.

语法

var x = touchItem.clientX;

返回值

x
触点相对于可见视区(visual viewport)左边沿的的X坐标. 不包括任何滚动偏移.
