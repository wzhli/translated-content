{{APIRef("HTML Drag and Drop API")}}

通过构造函数 DataTransfer 创建一个新的 {{domxref("DataTransfer")}} 对象。注意，单独创建该对象没有意义，且Internet Explorer中 DataTransfer 不是一个构造函数。

语法

var dataTrans = new DataTransfer()

参数

无

返回值

一个空的 DataTransfer 对象。
