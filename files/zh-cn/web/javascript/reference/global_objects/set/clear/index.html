{{JSRef}}

clear() 方法用来清空一个 Set 对象中的所有元素。

语法

mySet.clear();

返回值

{{jsxref("undefined")}}.

示例

var mySet = new Set();
mySet.add(1);
mySet.add("foo");

mySet.size;       // 2
mySet.has("foo"); // true

mySet.clear();

mySet.size;       // 0
mySet.has("bar")  // false
