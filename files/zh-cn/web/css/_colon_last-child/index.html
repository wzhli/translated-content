概述

:last-child CSS 伪类 代表父元素的最后一个子元素。

语法

{{csssyntax}}

示例

HTML 内容

<ul>
  <li>此元素背景色不是lime</li>
  <li>我的也不是lime。</li>
  <li>我的才是lime！ :)</li>
</ul>

CSS 内容

li:last-child {
  background-color: lime;
}
