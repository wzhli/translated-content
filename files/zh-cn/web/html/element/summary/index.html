概述

HTML <summary> 元素 用作 一个{{HTMLElement("details")}}元素的一个内容的摘要，标题或图例。

提示： 如果 <summary> 元素在浏览器内不能被正确解析和渲染，，则会展示<details>标签内的其他内容。
标签闭合 开始标签和结束标签都不能省略。
父级元素 {{ HTMLElement("summary") }}元素是{{ HTMLElement("details") }} 元素的子元素。
文档规范 HTML5, section 4.11.2

属性

本元素仅支持全局属性。

DOM 接口

本元素支持 {{domxref("HTMLElement")}} 接口。

示例

In summation, get details from the link on the next line

请参考 {{ HTMLElement("details") }} 页面内有关 <summary>元素的示例。
