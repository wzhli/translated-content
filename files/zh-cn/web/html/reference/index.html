HTML 是一种描述 Web 文档结构和语义的语言；它由元素组成，每个元素可以有一些属性。网页中的内容通过 HTML 元素标记，如{{HTMLElement("img")}}、{{HTMLElement("title")}}、{{HTMLElement("p")}}、{{HTMLElement("div")}} 等等。

本文提供了您进行 Web 开发时可以参考的资料。MDN 提供了组成 HTML 的所有元素的参考和每个属性的参考，这些属性可以控制元素的行为。

HTML 中的元素是有属性的；这些额外的属性值可以配置元素或者以各种方式来调整元素的行为，进而满足用户所需的标准。
