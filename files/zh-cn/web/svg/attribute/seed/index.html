seed属性代表了{{SVGElement("feTurbulence")}}元素生成的伪随机数的开始数字。

如果没有指定这个属性，它等效于指定了一个0值。

用法

类别 None
值 <number>
可变性 Yes
规范文档 SVG 1.1 (2nd Edition)

示例

元素

以下元素可以使用seed属性：

  • {{ SVGElement("feTurbulence") }}
