这个属性定义了一个动画的结束值，可以约束激活持续时间。

这个属性值是一个分号分隔的数列值。每个单独值可以是定义在{{ SVGAttr("begin") }}属性中同种类型。

用法

类别 动画定时属性
值 <End-value-list>
可变性 No
规范文档 SVG 1.1 (2nd Edition)

欲知在<end-value-list>中允许用哪些值，请阅读{{ SVGAttr("begin") }}属性。
