这个属性表示动画将发生的次数。

这个属性的值指定了重复的次数。它也可以包括用分数值表示。它的值必须大于0。

用法解释

类别 动画时间属性
Value <number> | "indefinite"
Animatable No
Normative document SVG 1.1 (2nd Edition)
