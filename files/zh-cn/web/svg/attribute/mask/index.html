mask属性绑定的元素将应用给定的{{ SVGElement("mask") }}元素。

作为一个外观属性，它可以直接用作CSS样式表内部的属性。

用法

类别 外观属性
值 <FuncIRI> | none | inherit
可变性 Yes
规范文档 SVG 1.1 (2nd Edition)
<funcIRI>
对同一SVG文档片段内部别的图形对象的一个引用 ，这个图形对象将用作遮罩。这个引用的句法与{{ cssxref("uri","CSS URI") }}的句法相同。
