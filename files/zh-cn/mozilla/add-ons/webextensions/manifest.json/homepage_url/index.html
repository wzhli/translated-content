{{AddonSidebar}}
值类型 字符串（String）
强制性 非强制
示例 
"homepage_url": "https://example.org/my-addon"

该扩展的主页地址。

如果 developer 键存在且包含“url”属性，它将会覆盖 homepage_url 键。

这是一个 localizable property.

示例

"homepage_url": "https://github.com/mdn/webextensions-examples/tree/master/beastify"
