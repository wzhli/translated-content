{{AddonSidebar}}
Type String
强制性 不强制
示例 
"short_name": "附加组件的短名称"

附加组件的短名称. 一般用在 name 特性太长的情况下. 建议短名称的长度不要超过12个字符. 如果没有在 manifest.json 主文件中声明 short_name, 并且 name 特性的值又比较长的话, 用户界面显示的时候可能只会显示一部分. 

关于本地化, 可查看 localizable property.

示例

"short_name": "附加组件的短名称"
