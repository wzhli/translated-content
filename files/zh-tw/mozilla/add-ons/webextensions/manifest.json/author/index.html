套件作者，用來顯示在瀏覽器的用戶介面中。如果有提供 developer 鍵而且裡面包含 "name" 屬性，那會覆蓋author鍵。不能指定多個作者。

這是一個可侷限的屬性。

 

範例

"author": "cool puppy"
